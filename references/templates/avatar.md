# Avatar Design

## Use Cases
- Personal social media profiles
- Team member avatars
- Brand mascots
- App user avatars
- Virtual characters

## Defaults
- Aspect ratio: 1:1 (Square)
- Style: Illustrated/Cartoon

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Subject | Yes | - | Avatar subject matter |
| Style | No | Cartoon | Visual style type |
| Features | No | - | Special characteristics |
| Color Tone | No | Warm | Primary color scheme |

## Style Presets

### Realistic Portrait
- Use for: Professional image, realistic avatar, personal brand
- Features: High fidelity, rich details, strong realism
- Template: `Create a realistic portrait avatar of {subject}. Render in photorealistic style with detailed facial features and {characteristic} appearance. Use professional studio lighting with {color} tones against a clean background. Center the composition in a 1:1 square ratio to achieve high quality studio photography aesthetic.`

### Cartoon/Illustrated
- Use for: Casual social, cute style, approachable image
- Features: Simplified features, vibrant colors, cartoonized
- Template: `Design a cute cartoon avatar of {subject} in an illustrated style. Simplify the features to convey a {characteristic} personality with vibrant {color} colors. Make the character friendly and approachable using clean vector art techniques. Center the portrait in a 1:1 square ratio with modern character design sensibilities.`

### Abstract/Artistic
- Use for: Creative expression, artistic style, unique personality
- Features: Artistic, abstract elements, creative expression
- Template: `Create an abstract artistic avatar representing {subject} through creative interpretation. Express a {characteristic} aesthetic using a {color} color palette. Incorporate geometric or organic shapes in a unique artistic style. Compose expressively in a 1:1 square ratio with a contemporary modern digital art approach.`

### Professional Headshot
- Use for: Professional networking, corporate image, formal occasions
- Features: Formal feel, professional atmosphere, business style
- Template: `Create a professional business avatar of {subject} in corporate style. Portray a {characteristic} appearance that conveys confidence and trustworthiness. Use {color} professional tones with a clean, polished finish in a formal atmosphere. Compose as a centered headshot in 1:1 square ratio with executive portrait quality.`

### Minimalist Icon
- Use for: Minimal style, iconic avatar, brand identity
- Features: Minimal design, iconified, high recognition
- Template: `Design a minimalist icon avatar of {subject} with simple, clean design principles. Create a {characteristic} representation using a {color} minimal color scheme. Apply geometric simplification to achieve an iconic style that is memorable and recognizable. Use flat design aesthetic in a 1:1 square ratio with modern logo-like quality.`

## Completion Rules
- **Subject not provided**: Prompt user to specify avatar subject
- **Style not specified**: Default to cartoon/illustrated style
- **Features not described**: Use generic friendly default features
- **Color tone not specified**: Smart recommendation based on style (realistic=natural, cartoon=vibrant, abstract=contrasting, business=neutral)
- **Aspect ratio**: Default 1:1 square, specify if other ratios needed
