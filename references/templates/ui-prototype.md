# UI Prototype Design

## Use Cases
- Low-fidelity wireframes
- Concept mockups
- Interface sketches
- Product prototypes
- Feature demonstrations

## Defaults
- Aspect ratio: Device-specific
  - Mobile: 9:19.5
  - Web/Desktop: 16:9
- Style: Modern clean

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Interface Type | Yes | - | UI type (Mobile/Web/Other) |
| Functionality | Yes | - | Core functionality description |
| Style | No | Modern clean | Visual style type |
| Fidelity | No | Mid-fidelity | Prototype detail level |

## Style Presets

### Mobile App
- Use for: iOS/Android apps, mobile interfaces, mobile-first design
- Features: Touch-friendly, clear information hierarchy, mobile platform compliance
- Template: `Create a mobile app UI prototype for {functionality}. Design using a modern {style} design system with a clean interface layout featuring {feature} screens. Include intuitive navigation patterns with thumb-friendly button placement and clear visual hierarchy. Apply appropriate spacing, padding, and mobile-first typography. Add status bar and navigation elements with a contemporary color scheme. Format in 9:19.5 portrait ratio following iOS/Android design guidelines for professional app mockup quality.`

### Web Dashboard
- Use for: Data display, admin panels, SaaS platforms, control panels
- Features: High information density, data visualization, modular functionality
- Template: `Design a web dashboard UI prototype for {functionality}. Create a {style} interface with sidebar navigation and top header layout. Include data visualization components for {feature} using card-based content modules. Add table and chart elements within a responsive grid system. Apply a professional color palette with clear information architecture and an admin panel aesthetic. Format in 16:9 landscape ratio with modern SaaS dashboard quality.`

### Landing Page
- Use for: Marketing pages, product promotion, event pages, conversion pages
- Features: Strong visual impact, conversion-oriented, scroll-based layout
- Template: `Design a landing page UI prototype for {functionality}. Create a {style} marketing design with a hero section featuring a compelling headline. Add clear call-to-action buttons and feature showcase sections for {feature}. Include social proof and testimonial areas within a responsive layout structure. Optimize for conversion with an attractive visual hierarchy following modern web design trends. Format in 16:9 viewport ratio for high-converting landing page quality.`

## Completion Rules
- **Interface type not provided**: Prompt user to specify mobile or web
- **Functionality not described**: Prompt user to describe core functionality or use case
- **Style not specified**: Default to modern clean style
- **Fidelity selection**:
  - Low-fidelity: Black and white wireframes, basic layout
  - Mid-fidelity (default): Basic colors, simple visual style
  - High-fidelity: Complete visual design, close to actual product
- **Aspect ratio auto-adapt**: Mobile defaults to 9:19.5 portrait, web defaults to 16:9 landscape
- **Component realism**: Include navigation bars, buttons, input fields, and other common UI component placeholders
