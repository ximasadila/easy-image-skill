# PPT Slides

## Use Cases
- Cover slides
- Section dividers
- Content illustrations
- Concept visualization
- Background images

## Defaults
- Aspect ratio: 16:9
- Style: Professional/Tech

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Theme | Yes | - | Main concept or theme |
| Style | No | Tech | Visual style preference |
| Color | No | Blue | Primary color scheme |
| Mood | No | Professional | Overall atmosphere |

## Style Presets

### Tech/Futuristic
- Use for: Tech products, digital transformation, AI/blockchain topics
- Features: Geometric patterns, gradients, light effects, digital elements
- Template: `Create a futuristic technology background for a presentation about {theme}. Design it with geometric patterns and holographic effects, using a gradient that transitions from {color1} to {color2}. Include modern digital interface elements with depth of field to create a high-tech atmosphere. Keep the composition clean and professional in 16:9 ratio.`

### Minimalist
- Use for: Corporate reports, simple presentations, text-heavy pages
- Features: Ample white space, simple lines, single focus
- Template: `Create a minimalist background for a presentation about {theme}. Design a clean, simple composition with generous white space and subtle geometric shapes. Use {color} as a restrained accent color. The overall feel should be elegant and professional with soft lighting and abstract elements in 16:9 ratio.`

### Corporate
- Use for: Annual reports, business proposals, formal occasions
- Features: Stable colors, symmetric composition, professional feel
- Template: `Create a professional corporate background for a presentation about {theme}. Design an elegant, balanced composition using a {color} color scheme with subtle textures. Include sophisticated design elements that convey trust and formality. The layout should be balanced and suitable for high-end business presentations in 16:9 ratio.`

### Creative
- Use for: Creative proposals, design showcases, brand stories
- Features: Bold colors, creative elements, artistic feel
- Template: `Create a creative artistic background for a presentation about {theme}. Use bold colors combining {color1} and {color2} in an imaginative composition with dynamic visual elements. The design should have artistic flair and inspire creativity while maintaining modern aesthetics in 16:9 ratio.`

### Natural Fresh
- Use for: Environmental topics, health products, education/training
- Features: Natural elements, soft colors, comfortable feel
- Template: `Create a fresh, nature-inspired background for a presentation about {theme}. Use soft {color} tones with organic elements and a clean, airy composition. The lighting should be gentle and calming, creating an approachable and friendly atmosphere in 16:9 ratio.`

## Completion Rules
- **Theme not provided**: Use "abstract professional" as default
- **Style not specified**: Default to tech/futuristic style
- **Color not specified**: Smart recommendation based on theme (tech=blue, business=dark, creative=colorful, nature=green)
- **Mood not specified**: Default to professional formal
- **Ratio requirements**: If specified (4:3 or vertical), adjust ratio parameter in prompt
