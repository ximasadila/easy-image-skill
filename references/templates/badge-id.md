# Badge/ID Design

## Use Cases
- Employee badges
- Event passes
- Membership cards
- Access cards
- Conference credentials

## Defaults
- Aspect ratio: 3:4 (Portrait orientation)
- Style: Corporate professional

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Purpose | Yes | - | Badge purpose type |
| Organization | No | - | Company/event name |
| Style | No | Corporate | Visual style type |
| Color | No | Brand colors | Primary color scheme |

## Style Presets

### Corporate Professional
- Use for: Company employee badges, business conferences, formal occasions
- Features: Formal and structured, clear readability, brand compliance
- Template: `Create a professional corporate badge for {purpose}. Design a clean, formal layout that incorporates {organization} branding and uses a {color} corporate color scheme. Include clear typography for name and title fields with a photo placeholder area at the top. Style this as a professional ID card with minimal decoration, ensuring high readability in a portrait 3:4 ratio with a business standard appearance.`

### Event/Conference
- Use for: Conference badges, exhibition passes, event tickets
- Features: Vibrant and engaging, clear theme, visually attractive
- Template: `Design a creative event badge for {purpose} that integrates the {organization} theme throughout. Use vibrant {color} colors with a dynamic layout featuring event branding and engaging visual elements. Include clear sections for attendee information in a lanyard-ready design. Make it eye-catching with modern conference pass aesthetics in a portrait 3:4 ratio.`

### Creative/Modern
- Use for: Tech companies, creative teams, young brands
- Features: Minimal and modern, strong design sense, trendy forward
- Template: `Create a modern minimalist badge for {purpose} featuring {organization} contemporary branding. Apply a clean {color} palette with geometric shapes and thoughtful use of negative space. Use sophisticated typography in a sleek information layout. Take a design-forward approach with subtle gradients or flat colors in a portrait 3:4 ratio that conveys premium ID card quality.`

- **Purpose not provided**: Prompt user to specify badge purpose (employee badge/event pass/membership card, etc.)

- **Style not specified**: Default to corporate professional style
- **Color not specified**: Smart recommendation based on style (corporate=neutral business colors, event=vibrant energetic colors, minimal=sophisticated premium colors)
- **Aspect ratio**: Default 3:4 portrait, can adjust to 4:3 for horizontal cards
