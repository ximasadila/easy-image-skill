# Introduction Graphic

## Use Cases
- Personal introduction cards
- Product introduction
- Company profile slides
- Team member introductions
- Service description graphics

## Defaults
- Aspect ratio: Varies by platform (16:9 for slides, 1:1 for social media)
- Style: Professional Friendly

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Subject | Yes | - | Introduction subject or theme |
| Type | No | Personal Bio | Introduction type |
| Style | No | Professional Friendly | Visual style preference |
| Color Tone | No | Warm tones | Primary color scheme |

## Style Presets

### Professional Bio
- Use for: Personal introductions, self-presentation, professional image
- Features: Person-focused, professional atmosphere, approachable
- Template: `Create a professional personal introduction graphic for {subject}. Design with a friendly, approachable atmosphere using a {color} color scheme. Keep the layout clean with space for text while maintaining a warm, professional feeling. Use modern personal branding style with a portrait-friendly composition suitable for bio cards.`

### Product Showcase
- Use for: Product introductions, feature explanations, feature showcase
- Features: Product prominence, visualized features, eye-catching
- Template: `Create a product introduction graphic for {subject} with a showcase-style composition. Apply {color} brand colors throughout a clean, modern design. Include feature highlighting elements with professional product photography aesthetic. Ensure a clear focal point and engaging visual style that is marketing-ready.`

### Team Intro
- Use for: Team members, organizational structure, personnel showcase
- Features: Team atmosphere, collaborative feel, unified style
- Template: `Create a team member introduction graphic for {subject}. Design with a collaborative atmosphere using a {color} consistent color scheme. Apply professional team aesthetic with an organized layout that feels friendly and accessible. Use modern corporate style with a cohesive design system suitable for member profiles.`

### Company Overview
- Use for: Enterprise introductions, company profiles, brand showcase
- Features: Corporate identity, brand presence, authority
- Template: `Create a company introduction graphic for {subject} that showcases corporate identity. Use a {color} brand color palette throughout a professional business atmosphere. Convey a trustworthy, established feeling with modern enterprise design. Keep the layout clean, sophisticated, and scalable with high-end presentation quality that aligns with brand aesthetic.`

### Service Description
- Use for: Service introductions, feature explanations, value showcase
- Features: Clear expression, visualized value, easy to understand
- Template: `Create a service introduction graphic for {subject} with a clear value proposition. Apply a {color} color scheme throughout a benefit-focused composition. Use easy-to-understand visual elements in professional service industry style. Design with a customer-friendly, modern, and approachable feel. Keep the layout informative with persuasive visual quality.`

## Completion Rules
- **Subject not provided**: Prompt user to specify the introduction subject
- **Type not specified**: Default to personal bio type
- **Style not specified**: Default to professional friendly style
- **Color tone not specified**: Smart recommendation based on type (personal=warm tones, product=brand colors, team=harmonious colors, company=business colors)
- **Aspect ratio**: Default 16:9, use 1:1 for social media, 9:16 for vertical screens
