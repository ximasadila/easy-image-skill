# Flowchart

## Use Cases
- Business processes
- User journeys
- Technical flows
- Decision trees
- Workflow demonstrations

## Defaults
- Aspect ratio: 16:9
- Style: Clean and Clear

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Theme | Yes | - | Flow theme or business scenario |
| Complexity | No | Medium | Flow complexity level |
| Style | No | Minimalist | Visual style type |
| Color Tone | No | Blue | Primary color scheme |

## Style Presets

### Minimalist
- Use for: Clear expression, professional documents, simple presentation
- Features: Clean lines, clear arrows, minimal decoration
- Template: `Create a minimalist flowchart for {theme}. Use clean lines and arrows with simple geometric shapes. Apply a {color} color scheme and establish clear hierarchy. Style this as a professional diagram with maximum clarity, minimal decoration, and organized layout in 16:9 ratio for a modern infographic design.`

## Completion Rules
- **Theme not provided**: Use "general process" as default theme
- **Complexity not specified**: Default to medium complexity (3-5 main steps)
- **Style not specified**: Default to minimalist style
- **Color tone not specified**: Smart recommendation based on usage (business=blue, technical=dark, creative=colorful, user=friendly colors)
- **Aspect ratio**: Default 16:9 horizontal, adjust to 9:16 for vertical flowcharts
