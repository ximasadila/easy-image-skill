# Report Illustration

## Use Cases
- Data visualization backgrounds
- Concept illustrations
- Report covers
- Statistical chart backgrounds
- Analysis presentation images

## Defaults
- Aspect ratio: 16:9
- Style: Clean Professional

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Theme | Yes | - | Report theme or data type |
| Style | No | Data Analytics | Visual style preference |
| Color Tone | No | Blue | Primary color scheme |
| Atmosphere | No | Professional | Overall atmosphere |

## Style Presets

### Data/Analytics
- Use for: Data reports, statistical analysis, business intelligence presentations
- Features: Chart elements, grid lines, data visualization motifs
- Template: `Create a professional data analytics background for {theme}. Design with abstract chart elements and grid patterns. Include data visualization motifs using a {color} color scheme with accent colors. Keep the composition clean and organized with a modern dashboard aesthetic. Add subtle infographic elements in 16:9 ratio with a business intelligence style.`

Make it inspiring and motivational in professional business style with 16:9 ratio.`

### Teamwork/Collaboration
- Use for: Team reports, collaboration projects, personnel-related reports
- Features: Connection elements, network patterns, collaborative atmosphere
- Template: `Create a teamwork and collaboration background for {theme}. Design with connected nodes and network patterns in a people-centric approach. Use a {color} color palette to establish a partnership atmosphere. Convey an inclusive and cooperative feeling with modern organizational style. ### Innovation/Technology
- Use for: Innovation projects, technical reports, digital transformation reports
- Features: Tech feel, cutting-edge elements, futuristic vibe
- Template: `Create an innovation and technology background for {theme}. Design with cutting-edge tech elements and digital transformation motifs. Compose innovatively in 16:9 ratio with an advanced technology style.`

## Completion Rules
- **Theme not provided**: Use "business report" as default theme
- **Style not specified**: Default to data/analytics style
- **Color tone not specified**: Smart recommendation based on report type (data=blue, growth=green, team=orange, tech=purple-blue)
- **Atmosphere not specified**: Default to professional formal atmosphere
- **Aspect ratio**: If special requirements (A4 portrait or square), adjust ratio parameter in prompt
