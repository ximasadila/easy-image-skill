# Scene Photo

## Use Cases
- Lifestyle shots
- Usage scenarios
- Atmosphere creation
- Background materials
- Brand storytelling

## Defaults
- Aspect ratio: 4:3 or 16:9 (horizontal scenes)
- Style: Natural/Authentic
- Lighting: Natural light feel

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Scene | Yes | - | Scene type and environment |
| Mood | No | Cozy | Emotional atmosphere |
| Time | No | Daytime | Time setting affects lighting |
| Elements | No | Natural | Key elements in scene |
| Angle | No | Eye-level | Camera perspective |

## Style Presets

### Indoor Cozy
- Use for: Home life, coffee time, reading spaces, home office
- Features: Warm tones, soft lighting, comfortable feel, lived-in
- Template: `Create a cozy indoor photograph of {scene}. Capture a warm, inviting atmosphere with soft natural light streaming through windows. Use {color} warm tones throughout to convey comfort and homeliness. Include lived-in details and gentle shadows that create a peaceful ambiance. Compose the shot intimately to feel authentic and welcoming. Style this as lifestyle photography in 4:3 or 16:9 ratio.`

## Completion Rules
- **Scene not provided**: Request user to specify - required field
- **Mood not specified**: Smart recommendation based on scene type
- **Time not specified**: Daytime soft light for indoors, golden hour for outdoors
- **Elements not specified**: Add appropriate elements based on scene
- **Angle not specified**: Wide eye-level for spaces, slightly elevated for details
- **Ratio selection**: 16:9 for spacious scenes, 4:3 for cozy intimate scenes
