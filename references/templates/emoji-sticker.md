# Emoji/Sticker Design

## Use Cases
- WeChat stickers
- Slack custom emoji
- Chat reaction images
- Brand custom emotes
- Social media stickers

## Defaults
- Aspect ratio: 1:1 (Square)
- Style: Cute/Kawaii
- Background: Transparent or simple solid

## Core Elements
| Element | Required | Default | Description |
|---------|----------|---------|-------------|
| Emotion | Yes | - | Emotion/action to express |
| Subject | No | Simple character | Sticker subject matter |
| Style | No | Cute cartoon | Visual style type |
| Usage | No | General chat | Usage context |

## Style Presets

### Cute/Kawaii
- Use for: Daily chat, close relationships, relaxed atmosphere
- Features: Big eyes, rounded shapes, warm and healing
- Template: `Create a cute kawaii sticker expressing {emotion}. Design an adorable {subject} character with big expressive eyes and rounded, soft shapes. Keep the design simple and clean with minimal details and vibrant, cheerful colors. Use a transparent or pastel background with centered composition in 1:1 square ratio. Make it perfect for messaging apps with a charming, lovable style.`

### Meme-style
- Use for: Humor and jokes, internet culture, fun expressions
- Features: Exaggerated expressions, humor, trending memes
- Template: `Design an expressive meme sticker showing {emotion}. Feature {subject} with an exaggerated reaction that feels humorous and relatable. Create bold expressions with clear visual impact using high contrast colors. Keep the background simple or transparent so it reads easily at small size. Make it in 1:1 square ratio with viral meme quality that's fun and shareable.`

### Brand Mascot
- Use for: Corporate branding, product image, marketing promotion
- Features: Brand recognition, strong consistency, professional yet cute
- Template: `Create a brand mascot sticker with a {emotion} expression. Design the {subject} character to represent brand identity with consistent style and personality. Make it professional yet friendly with recognizable design elements in a clean vector style. Use on-brand colors and aesthetic with a transparent or subtle background. Ensure it's versatile for multiple uses in 1:1 square ratio with polished commercial quality.`

## Completion Rules
- **Emotion not provided**: Prompt user to specify the emotion or action to express
- **Subject not specified**: Default to simple cute round character or universal emoji symbol
- **Style not specified**: Default to cute/kawaii style
- **Background handling**: Prefer transparent background, or select simple solid/gradient based on style
- **Size consideration**: Design must be clear and readable even when scaled down to chat emoji size
- **Text usage**: If text needed, keep it short (1-3 characters/words), with clear readable font
